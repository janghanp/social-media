## Features

- A user can edit photos before posting.
- Images are stored in Amazon S3 bucket.
- Authentication with NextAuth.js.
- Infinite scroll.

## Tech Stack

- **Typescript**
- **Nextjs12 (without app router)**
- **Tailwindcss (daisyui)**
- **zustand & SWR & React Context**
- **Prisma**
- **MongoDB**
